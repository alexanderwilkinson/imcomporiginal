# Image Comparator (imcomp)

`imcomp` is a standalone tool to match corresponding regions in a pair of images.
It is an open source project developed at the [Visual Geometry Group](http://www.robots.ox.ac.uk/~vgg/) and 
released under the BSD-2 clause license.
